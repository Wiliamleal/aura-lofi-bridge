# Aura Lofi Bridge Server - Netlify

Este é um projeto separado apenas para hospedar as Netlify Functions do bridge server.

## Deploy

1. Criar novo repositório GitHub chamado `aura-lofi-bridge`
2. Fazer push deste diretório
3. Conectar no Netlify como novo site
4. Configurar variáveis de ambiente

URL esperada: `https://aura-lofi-bridge.netlify.app`
